Informática

O Centro Federal de Educação Tecnológica de Minas Gerais (CEFET-MG) é uma autarquia federal brasileira, vinculada ao Ministério da Educação, que oferece ensino médio, cursos técnicos, superiores, pós stricto sensu e lato sensu, contemplando também, de forma indissociada, o ensino, a pesquisa e a extensão, na área tecnológica e no âmbito da pesquisa aplicada.

Conteúdo

Lógica e introdução a programação

Durante o primeiro ano no cefet é introduzido a lógica de programação, tais como estrutura de repetição, condicionais etc. Primeiro com linguagens de alto nível com alto grau de abstração e fácil entendimento, voltado para quem nunca teve contato com tais ferramentas e tecnologias. Durante o decorrer dos dois últimos anos, focamos no desenvolvimento orientado a objetos POO, estudando sobre como estruturar melhor o código. Foi introduzido alguns padrões de projetos e boas práticas na hora de codificar.

Desenvolvimento WEB

Durante os três anos tivemos contato com desenvolvimento WEB utilizando HTML, CSS, JavaScript e PHP. Utilizamos também em alguns projetos a framework Bootstrap para agilizar o processo de estilização de alguns componentes. Tivemos contato também no terceiro ano com a framework Laravel, framework essa baseada em PHP e muito utilizada no mercado atualmente.

Desenvolvimento Mobile

No terceiro ano tivemos a matéria de desenvolvimento em android nativo, utlizando java e xml. Como IDE utilizamos o Android Studio, ferramenta essa que tem um grande suporte para o desenvolvimento android.

Outros

Além de lógica de programação, tivemos contato com banco de dados e consequentemente com SQL. Vimos as diferenças entre banco de dados relacionais e não relacionais, trabalhamos com MySql e MongoDB. Além disso utilizamos os sistemas operacionais Windows e linux para desenvolver
